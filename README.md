# Recipe-website
🍽 Recipe Explorer Web App

This project is a responsive recipe browsing web application developed using HTML, CSS, and JavaScript. It allows users to explore different types of recipes categorized into sections such as Easy, Breakfast, Kids, Dessert, Millet, Oil-Free, and Junk Food.

The application displays recipe cards containing the dish name, cooking time, ingredients, and preparation steps. Users can click the “View Recipe” button to expand and see the complete details of the dish.

The project also includes interactive features such as search filtering, star rating, comments for each recipe, and a simple login/register system using localStorage. Users can search for recipes quickly, rate dishes, and leave comments to share their feedback.

This project demonstrates the use of DOM manipulation, event handling, dynamic rendering, and client-side data storage in JavaScript while creating an engaging and user-friendly cooking guide interface.

🛠 Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

LocalStorage (for login, ratings, and comments)

✨ Features

Recipe categories with multiple dishes

Expandable View Recipe dropdown

Search bar with debounce filtering

Star rating system for recipes

Comment section for user feedback

Simple login/register system

Scroll-to-top button
